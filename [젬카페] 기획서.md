## 2. 기획 의도

### 2.1 문제 정의

SNS 마케팅은 카페·디저트 업종 소상공인에게 사실상 필수 영업 채널로 자리 잡았다. 인스타그램·유튜브 쇼츠 등 숏폼 영상 플랫폼의 영향력이 절대적임에도 불구하고, 대다수 소상공인은 다음과 같은 이유로 영상 콘텐츠 제작에 어려움을 겪고 있다.

| 문제 | 내용 |
|---|---|
| 비용 | 전문 제작사 의뢰 시 영상 1편당 수십만 원 수준 |
| 시간 | 자체 촬영·편집 시 수 시간 이상 소요 |
| 기술 | 영상 편집 도구 사용에 전문 지식 필요 |
| 지속성 | 신메뉴 출시마다 새 콘텐츠 제작이 요구되나 내부 역량 부족 |

### 2.2 서비스 방향

gem.cafe는 위 문제를 해소하기 위해 기획된 **AI 기반 영상 자동 생성 서비스**이다. 사용자는 케이크 사진을 업로드하고 간단한 옵션을 선택하는 것만으로 수 분 내에 완성도 높은 SNS 광고 영상을 얻을 수 있다.

- AI가 케이크 구성 성분(베이스·크림·토핑·코팅)을 자동 분석하여 최적의 시뮬레이션을 추천한다.
- 생성된 영상은 인브라우저 편집기에서 텍스트·스티커·BGM을 추가한 후 YouTube·Instagram에 직접 업로드할 수 있다.
- 모든 다운로드 영상에는 포렌식 워터마크가 자동 삽입되어 콘텐츠 출처를 보호한다.

## 3. 주요 기능

### 3.1 AI 영상 생성 마법사

케이크 사진 업로드부터 최종 영상 생성까지 8단계로 사용자를 안내하는 핵심 플로우이다.

```
1단계  메뉴 사진 업로드
  ↓
2단계  AI 성분 분석 (Moondream3 — 베이스 · 크림 · 토핑 · 코팅 키워드 추출)
  ↓
3단계  강조 키워드 선택 (분석 결과 칩 형태로 제공)
  ↓
4단계  시뮬레이션 선택 (키워드 카테고리별 필터링, 총 7종)
  ↓
5단계  배경 선택 (6종)
  ↓
6단계  한국어 영상 프롬프트 생성 (Gemini 자동 생성 후 사용자 편집 가능)
  ↓
7단계  키프레임 이미지 생성 (nano-banana-pro/edit, 최대 3회 시도)
  ↓
8단계  5~6초 광고 영상 생성 (Veo 3.1 first-last-frame)
```

**시뮬레이션 분류**

| 카테고리 | 시뮬레이션 종류 |
|---|---|
| 베이스(시트) | 한 조각 들어올리기, 포크로 한입 뜨기, 칼로 단면 가르기 |
| 크림 | 한 스푼 떠내기, 뭉개기, 손으로 반 가르기 |
| 토핑 | 위에서 떨어트리기 |

**배경 종류**

흰 대리석 / 카페 인테리어 / 야외 정원 / 원목 테이블 / 미니멀 화이트 / 어두운 무드

### 3.2 세션 자동 저장 및 복원 (이어서 만들기)

영상 생성 도중 세션이 종료되더라도 진행 상태가 서버에 자동 저장된다. 재접속 시 강조 키워드·시뮬레이션·배경·프롬프트·키프레임 선택값이 복원되어 중단 시점부터 작업을 재개할 수 있다.

### 3.3 풀스크린 영상 편집기

AI 생성 영상에 부가 요소를 합성하는 인브라우저 캔버스 기반 편집기이다.

| 기능 | 상세 |
|---|---|
| 텍스트 오버레이 | 한글 폰트 6종, 크기 프리셋 5단계, 15색 팔레트, Bold/Italic/Outline/회전 지원 |
| 스티커 오버레이 | Twemoji 기반 93개 이모지, 8개 카테고리, 드래그·리사이즈·회전 |
| BGM | Audius API 연동, 카테고리별 음원 검색·미리듣기, 볼륨 조절, 원본음 믹싱 |
| 내보내기 | MediaRecorder API → MP4 인코딩 → 워터마크 삽입 → 저장 |

### 3.4 영상 배포

| 기능 | 상세 |
|---|---|
| 다운로드 | DCT 포렌식 워터마크 삽입 후 로컬 저장. 최초 생성된 워터마크 파일은 세션 내 캐시하여 재요청 없이 즉시 다운로드 |
| 공유 | Web Share API 활용. 워터마크 영상 파일 및 링크 공유. 브라우저 미지원 시 링크 단독 공유 |
| SNS 자동 업로드 | YouTube · Instagram 계정 연동. 제목·설명·태그·캡션 입력 후 자동 게시. SSE로 진행 상태 수신, 완료 시 PWA 푸시 알림 발송 |

### 3.5 PWA

| 기능 | 상세 |
|---|---|
| 앱 설치 | Android: `beforeinstallprompt` 이벤트 캡처 후 인앱 설치 배너 표시. iOS: 홈 화면 추가 수동 안내 |
| 업데이트 알림 | 새 Service Worker 감지 시 토스트 노출. 사용자 확인 후 즉시 적용 |
| 푸시 알림 | 영상 생성·워터마크 처리 완료 시 백그라운드 Web Push 발송. 알림 클릭 시 해당 영상 상세 페이지로 이동 |

### 3.6 인증

| 방식 | 상세 |
|---|---|
| 이메일/비밀번호 | 회원가입 및 로그인. 클라이언트 유효성 검사 포함 |
| Google OAuth 2.0 | Google Identity Services SDK 활용. 신규 사용자는 전화번호 추가 입력 단계로 연결 |
| 토큰 관리 | JWT accessToken + refreshToken (HttpOnly 쿠키). axios 인터셉터로 만료 시 자동 갱신 |

## 4. 사용 외부 API

| 서비스 | 제공사 | 용도 |
|---|---|---|
| **Moondream3** | fal.ai | 케이크 이미지 성분 분석 (베이스·크림·토핑·코팅 키워드 추출) |
| **Gemini 2.5 Flash-Lite** | SSAFY GMS (Google) | 한국어 영상 프롬프트 자동 생성 및 한→영 번역 |

| **Veo 3.1 first-last-frame** | fal.ai | I2V(Image-to-Video) — 시작·마지막 프레임 기반 5~6초 영상 생성 |
| **Google Identity Services** | Google | Google OAuth 2.0 소셜 로그인 |
| **Audius API** | Audius | 저작권 프리 BGM 스트리밍 및 카테고리별 음원 검색 |
